Tags Function
<h1> - <h6> Headers 1 - 6 it controls how important is the line written inside those tags. The bigger the number the smaller is the font. we can also edit the font details using CSS. Personally i use the numbers to easily navigate the heading and edit itright away.
<Article> This is the Article tag. Writing using this tag can help us write an article in our web-page. personally i think this is a perfect tag for a bloger web-pages.
<table> This allows us to creat a table within our web-page. tables can help us organise the page to multiple zones.

Tags Combinations.

Combining more than one tag togther can help the designer keep their code easier to read and imagine how it going to look like when it is combiled. so it is important.

Some combinations must happen for the code to work, like Table tags. A Table tag needs a row tag <tr> reqired to creat a row inside our table. thats one example of many.
